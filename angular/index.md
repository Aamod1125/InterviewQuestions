### How to install & Also how to install specific version of the angular

Install :
```
install node
npm install -g @angular/cli
ng new my-first-app
npm start

```
Install specific version:
```
npm uninstall -g @angular/cli
npm cache clean 
npm install -g @angular/cli@17.1.0

```
